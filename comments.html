主文档页面都使用了什么技术？

主文档和该说明文档一样，均使用纯 HTML + CSS 编写而成，并未使用 Javascript 脚本。其中部分动画效果（鼠标悬浮使得示意图中的链接颜色出现些许变化）均使用 CSS 实现。

示意图由 Microsoft Powerpoint 画成，导出为svg格式，经修改后直接嵌入到网页中（因为需要添加部分文字链接）。

整体页面样式由 Jason Long 的 Cayman theme 提供。而上述动画效果是自己写的。


参考资料是怎么选择的？

教程和参考类的资料大都是我所用过的。而涉及某一具体点的资料，是我认为写得好的，对学习某个点有用的资料。

文案是你自己写的吗？HTML 是你自己写的吗？

都是我自己写的，HTML 也是自己写的，用 VS Code 写成的。我自己用的是 Chrome 浏览器，如果有什么排版上的问题，那请使用 Chrome 浏览器观看。
